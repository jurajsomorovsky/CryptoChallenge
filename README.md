# CryptoChallenge
Dobbertin Challenge 2012 whose aim was to exploit backwards compatibility attacks

## Prerequisites
Java 8 and maven

## HowTo
Compile with:
```
mvn clean package
```

Run server with:
```
java -jar target/CryptoChallenge-1.0-SNAPSHOT.jar
```

You can then see the instructions at: `http://localhost:50080/`
